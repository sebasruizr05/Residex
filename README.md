# Residex
Plataforma de gestión inteligente para unidades residenciales

## 🚀 Descripción

Residex es una solución web que permite centralizar la gestión de unidades residenciales, incluyendo:

- Control de visitantes
- Gestión de paquetería
- Administración de pagos
- PQRS (incidencias)
- Anuncios y comunicación
- Reservas de zonas comunes

## 🏗️ Arquitectura

El sistema está basado en una arquitectura en capas desplegada en AWS:

- Frontend: Next.js
- Backend: NestJS
- Autenticación: Amazon Cognito
- API: API Gateway
- Base de datos: PostgreSQL (RDS)
- Almacenamiento: Amazon S3
- CDN: CloudFront
